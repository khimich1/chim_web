## Контекст продукта

**chim_web** — платформа репетитора по химии для работы с учениками (ЕГЭ/ОГЭ): ДЗ, учебник с аудио, пошаговые тесты, рейтинг, AI-советчик для преподавателя.

**Текущая монетизация офлайн:**
- Индивидуально: 2000 ₽/час
- Группа 4 чел.: 1500 ₽/чел. × 2 часа

**Стратегическое решение:** веб усиливает занятия и приводит лидов (SEO), а не заменяет репетитора на старте.

## 1. SWOT

### Strengths (сильные стороны)

| # | Фактор |
|---|--------|
| S1 | Многолетний опыт преподавания и доверие учеников |
| S2 | Собственный контент: учебник, тесты ЕГЭ/ОГЭ, аудио |
| S3 | Рабочий продукт (FastAPI + Next.js), не «слайды» |
| S4 | AI-советчик с просмотром диалогов преподавателем |
| S5 | Геймификация: рейтинг, прогресс, streak |
| S6 | Прямой доступ к ЦА для custdev |
| S7 | Модульная архитектура (треки ege/oge, слои API) |

### Weaknesses (слабые стороны)

| # | Фактор |
|---|--------|
| W1 | Нет публичного онбординга и SEO-витрины (на момент discovery) |
| W2 | Один преподаватель = узкое горлышко масштаба |
| W3 | Нет оплаты в продукте |
| W4 | Нет кабинета родителя |
| W5 | Бренд неизвестен вне текущей базы |
| W6 | Solo-dev: скорость фич vs конкуренты |

### Opportunities (возможности)

| # | Фактор |
|---|--------|
| O1 | Стабильный спрос на подготовку к ЕГЭ/ОГЭ |
| O2 | Родители хотят контроль прогресса |
| O3 | SEO по long-tail («задача 28 ЕГЭ химия») |
| O4 | AI в EdTech — мало качественных решений по химии |
| O5 | Гибрид офлайн + digital после пандемии |
| O6 | Контент учебника → SEO и цифровые продукты |
| O7 | Группы — выше выручка на час (6000 ₽ vs 2000 ₽) |

### Threats (угрозы)

| # | Фактор |
|---|--------|
| T1 | Крупные игроки: Умскул, Фоксфорд, агрегаторы |
| T2 | Бесплатный контент: YouTube, Telegram |
| T3 | Низкая вовлечённость между занятиями |
| T4 | ПДн несовершеннолетних |
| T5 | AI как «списывание» — репутационный риск |
| T6 | Сезонность (пик осень–весна) |
| T7 | Копирование контента |

### Стратегические связки

| Связка | Действие |
|--------|----------|
| S1+S2+O3 | SEO-сайт на своём контенте → лиды |
| S4+O4 | AI как помощник между занятиями, не замена |
| W1+W2 | Онбординг + маркетинговая витрина (в работе) |
| T5 | Логи AI для преподавателя (уже в продукте) |
