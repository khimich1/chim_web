## 1. Решение: два сайта, одна воронка

### Рекомендация (зафиксировать)

| Слой | Домен (пример) | Стек | Назначение |
|------|----------------|------|------------|
| **Маркетинг + SEO** | `himiya-ege.ru` / `chim-tutor.ru` | Отдельный Next.js (static/SSG) или Tilda → позже Next | Привлечение: родители и ученики из поиска |
| **Приложение** | `app.himiya-ege.ru` | Текущий chim_web (Next.js + FastAPI) | Работа учеников: ДЗ, тесты, учебник — **noindex** |

**Почему не один сайт:**
- App требует авторизации, cookie, личные данные — плохая SEO-цель.
- Маркетинговые страницы нужны быстрые, с мета-тегами, schema.org, блогом — без логики кабинета.
- Разделение снижает риск «утечки» закрытого контента в индекс.

**Почему не только Tilda/WordPress:**
- У вас уже есть Next.js-экспертиза и контент в app — со временем блог можно на Next SSG с тем же репозиторием (monorepo) или отдельным `marketing/`.

### Воронка

```
[Яндекс/Google: «репетитор химия егэ»]
        ↓
[SEO-страница: услуга / статья / бесплатный тест]
        ↓
[CTA: Записаться на диагностику] → форма / Telegram / WhatsApp
        ↓
[Пробное занятие] → создание аккаунта в app
        ↓
[Онбординг в app] → удержание между занятиями
```

## 2. Цели SEO

| Цель | Метрика | Горизонт |
|------|---------|----------|
| Лиды на занятия | ≥5 заявок/мес | 4–6 мес после старта |
| Органический трафик | 500+ визитов/мес | 6–9 мес |
| Эфф. стоимость лида (органика) | эквивалент < 1 500 ₽ (vs платная реклама 2 500–4 000 ₽ CPL) | При стабильном потоке |
| Поддержка freemium (фаза 2) | Регистрации на бесплатный тест | 9+ мес |

**Unit-экономика (ориентир):**
- 1 новый ученик с SEO → индивидуально 2000 ₽/ч × 4 занятия/мес ≈ 8000 ₽/мес.
- Группа 4 чел. × 1500 ₽ × 2 ч = 12000 ₽/занятие.
- Окупаемость SEO при 10+ статьях и 0 затратах на разработку (своё время).

## 3. Целевая аудитория поиска

### Primary: Родитель

| Job | Пример запроса |
|-----|----------------|
| Найти репетитора | `репетитор по химии егэ [город]`, `репетитор химия 11 класс` |
| Оценить качество | `отзывы репетитор химия`, `как выбрать репетитора егэ` |
| Понять проблему ребёнка | `ребёнок не понимает химию`, `как подготовиться к егэ по химии` |

### Secondary: Ученик

| Job | Пример запроса |
|-----|----------------|
| Разобрать задачу | `задача 28 егэ химия разбор`, `расчёты по химии егэ` |
| Потренироваться | `тесты егэ химия онлайн`, `варианты егэ химия с ответами` |
| Выучить тему | `электролиз егэ химия`, `органика егэ таблица` |

**Стратегия контента:** 60% запросы родителей (деньги) + 40% ученические long-tail (трафик и доверие).

## 6. Переиспользование контента из app

| В app (закрыто) | На SEO (открыто) |
|-----------------|------------------|
| Темы учебника | Статья-конспект без аудио / урезанная версия |
| Варианты тестов | 5 вопросов бесплатно, полный банк — после записи |
| AI-советчик | Не выносить; УТП только на занятиях |
| Рейтинг | Не выносить |

**Важно:** не индексировать полные варианты ЕГЭ — иначе нет причины заходить в app.

## 7. Технический SEO (чеклист)

### Маркетинговый сайт

- [ ] `sitemap.xml`, `robots.txt`
- [ ] Уникальные `title` / `description` на каждой странице
- [ ] Open Graph для соцсетей
- [ ] Schema.org: `Person` / `LocalBusiness` (если применимо), `Course`, `FAQPage`
- [ ] Core Web Vitals: LCP < 2.5s (SSG, оптимизация изображений)
- [ ] HTTPS, canonical URLs
- [ ] Яндекс.Вебмастер + Google Search Console
- [ ] Яндекс.Метрика / GA4: цели на «отправку формы», «клик Telegram»

### App

- [ ] `robots.txt`: `Disallow: /student`, `/teacher`, `/login`
- [ ] `<meta name="robots" content="noindex">` на всех страницах кабинета
- [ ] Не ссылаться на закрытые URL с маркетинга без авторизации

### Домен и хостинг

| Вариант | Плюсы | Минусы |
|---------|-------|--------|
| **A. Subdomain** `app.domain.ru` | Чистое разделение | Два деплоя |
| **B. Monorepo** `apps/marketing` + `apps/web` | Один репозиторий | Сложнее CI |
| **C. Tilda + app subdomain** | Быстрый старт маркетинга | Слабее кастом, vendor lock |

**Рекомендация для старта:** **C** (Tilda/Readymag на 1–2 мес) + текущий app на subdomain **ИЛИ** **B** если готовы потратить 1–2 недели на `marketing/` в Next.js.

## 9. Конкуренты и позиционирование в поиске

| Тип конкурента | Примеры | Как отличаться в SEO |
|----------------|---------|----------------------|
| Агрегаторы | Профи.ру, Авито | Личный сайт + экспертный контент + «живой» преподаватель |
| Онлайн-школы | Умскул, Фоксфорд | Не масштаб, а **индивидуальный подход + своя платформа** |
| Банки заданий | Решу ЕГЭ | Разборы с объяснением + связка с занятиями |
| YouTube | Борщов, химия каналы | Текст + интерактивный тест + репетитор |

### УТП для заголовков (H1/meta title)

> Репетитор по химии ЕГЭ/ОГЭ — [N] лет опыта · Личная платформа подготовки · [Город/онлайн]

Не копировать «лучший репетитор» — конкретика: опыт, формат, платформа, результаты.

## 10. Риски SEO

| Риск | Вероятность | Митигация |
|------|-------------|-----------|
| Долгий выход в топ (3–6 мес) | Высокая | Параллельно сарафан + Telegram |
| Низкая конверсия трафика | Средняя | Сильный CTA, отзывы, цены на сайте |
| Каннибализация занятий бесплатным контентом | Средняя | Полный банк и AI — только в app |
| Нет времени на статьи | Высокая | 1 статья/нед, переиспользовать конспекты учебника |
| Регион не указан | Средняя | Явно указать город + «онлайн по всей России» |

## Резюме решения

1. **Два сайта:** маркетинг (индексируем) + app (noindex).
2. **Цель SEO — лиды на занятия**, не подписка на старте.
3. **Первые 90 дней:** 4 коммерческие страницы + 10 статей + 1 бесплатный мини-тест.
4. **Контент из учебника** — главный рычаг без дублирования полного продукта.
5. **Custdev с родителями** — уточнить реальные поисковые запросы до масштабирования контента.
