# Шаблоны SEO-статей для chim_web

**Связано с:** [SEO-стратегия](../strategy/seo-strategy.md)  
**Дата:** 19.06.2026

Используйте темы из учебника (`/api/textbook/topics`) как основу. Одна тема учебника → одна SEO-статья (урезанная версия без аудио и полного курса).

---

## Шаблон A: Разбор задания ЕГЭ (long-tail)

**Цель:** трафик учеников + лиды на диагностику  
**Объём:** 1500–2200 слов  
**Пример URL:** `/blog/zadacha-28-ege-himiya-raschety`

### Meta

```
Title: Задача 28 ЕГЭ химия: разбор и алгоритм решения | [Имя репетитора]
Description: Пошаговый разбор задания 28 ЕГЭ по химии: расчёты по уравнению реакции. Примеры, типичные ошибки и бесплатная диагностика.
```

### Структура

```markdown
# [Номер] задание ЕГЭ по химии: [краткое название темы]

## Краткий ответ
[2–3 предложения — для featured snippet]

## Что проверяет это задание
- Навык 1
- Навык 2

## Алгоритм решения (5 шагов)
1. ...
2. ...

## Пример из варианта [год/номер]
**Условие:** ...
**Решение:** ...
**Ответ:** ...

## Типичные ошибки
| Ошибка | Почему возникает | Как избежать |
|--------|------------------|--------------|

## Как потренироваться
[CTA: бесплатный мини-тест — 5 задач на эту тему]

## Нужна помощь с химией?
[CTA: запись на бесплатную диагностику 30 мин — Telegram / форма]
```

## Шаблон B: Тема из учебника (конспект)

**Цель:** экспертность + SEO по темам школьной программы  
**Объём:** 2000–2500 слов  
**Пример URL:** `/blog/elektroliz-ege-himiya`

### Meta

```
Title: Электролиз для ЕГЭ по химии: теория и задачи | [Имя]
Description: Конспект по электролизу для подготовки к ЕГЭ: определения, правила, примеры задач. От репетитора с [N] лет опыта.
```

### Структура

```markdown
# [Тема]: подготовка к ЕГЭ по химии

## Зачем это нужно на экзамене
[Связь с номерами заданий ЕГЭ]

## Ключевые определения
- **Термин 1** — ...
- **Термин 2** — ...

## Основные правила / закономерности
1. ...
2. ...

## Примеры
### Пример 1 (базовый)
### Пример 2 (уровень ЕГЭ)

## Частые вопросы учеников
### Вопрос 1?
Ответ...

## Шпаргалка (кратко)
| ... | ... |
```

## Чеклист перед публикацией

- [ ] Уникальный title и description (не дублировать другие страницы)
- [ ] H1 один на странице, содержит ключевое слово
- [ ] 2+ внутренние ссылки (блог ↔ услуги ↔ цены)
- [ ] CTA в середине и в конце
- [ ] Изображение с alt-текстом (схема, формула)
- [ ] Schema.org: `Article` или `FAQPage` где уместно
- [ ] Не публиковать полные варианты ЕГЭ из app

## Пример заполненной статьи (фрагмент, Шаблон A)

**CTA (середина):**  
> Не уверены в уровне? Пройдите [бесплатный мини-тест из 5 задач на расчёты](/besplatno/diagnosticheskiy-test) или запишитесь на [диагностику с репетитором](/zapis).
